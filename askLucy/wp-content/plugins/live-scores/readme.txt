= Features =

The wordpress plugin 'Live Scores' is covering up to 7 sport events in more than 250+ countries currently under 4 Languages. (EN/IT/FR/ES) - We will be adding more languages every week!

* American Football ( NFL & CFL )
 
* Basketball
 
* Handball
 
* Ice Hockey
 
* Soccer / Football
 
* Tennis ( ATP & WTA )
 
* Volleyball

== Installation ==

1. Upload the `live-scores` folder to the `/wp-content/plugins/` directory
1. Activate the Live Scores through the 'Plugins' menu in WordPress
1. Configure the plugin by going to the `Live Scores` menu that appears in your admin menu
